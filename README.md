# How to use

### Step 1

Change the variables:

You need to find this variables in main.py and change value.Use your personal values. How to find or create them you can find on the Internet.

![image](https://github.com/Aleksey767/apple_system_status_to_slack/assets/98593351/d116b716-b969-41ec-adfd-2e9a825fb3bd)

Also you can change this time. In my case, i'am checking apple site one time in 10 minutes.

![image](https://github.com/Aleksey767/apple_system_status_to_slack/assets/98593351/816a0be0-8533-4261-a3b9-26aa672693cc)
